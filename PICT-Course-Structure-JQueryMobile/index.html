PICT B.E. Course Structure

First Year (FE)

Engineering Mathematics-I
Fundamentals of calculus, matrices, and differential equations.
Engineering Physics
Concepts of mechanics, optics, and modern physics applications.
Engineering Chemistry
Basics of chemical bonding, thermodynamics, and engineering materials.
Programming and Problem Solving
Introduction to C programming and problem-solving techniques.
Basic Electrical Engineering
Foundations of electrical circuits and systems.

Second Year (SE)

Data Structures and Algorithms
Core data structures, algorithms, and their applications.
Computer Organization and Architecture
Basics of computer hardware and processor design.
Object Oriented Programming
OOP principles using Java and application development.
Database Management Systems
Database design, SQL queries, and normalization concepts.
Discrete Mathematics
Set theory, logic, combinatorics, and graph theory fundamentals.

Third Year (TE)

Operating Systems
Concepts of process management, memory management, and file systems.
Web Technologies
Frontend and backend development using modern web stacks.
Artificial Intelligence
Introduction to AI concepts, search algorithms, and knowledge representation.
Computer Networks
Networking basics, TCP/IP, and network security principles.
Software Engineering
Software development lifecycle, agile models, and project management.

Final Year (BE)

Machine Learning
Supervised and unsupervised learning, model evaluation, and ML applications.
Information and Cyber Security
Principles of cybersecurity, cryptography, and ethical hacking.
Blockchain Technology
Basics of blockchain architecture, cryptocurrencies, and smart contracts.
Big Data Analytics
Handling and analyzing large-scale data using modern tools.
Project Work
Capstone project applying IT skills to real-world problems.
